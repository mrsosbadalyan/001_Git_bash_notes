Here are the most used commands in gitbash

git log - see previous saves
q - exit after git log
THE BELLOW PART IS USED TO CREATE BRANCHES AND WORK ON A CODE, THEN COME BACK TO PREVIOUS VERSION ON GLOBAL PART git checkout /old version code/ - returns the previous versions
git checkout /old version code/ -f - force return if no working
git branch - checking the branches
git checkout master -taking back to master
git branch /new name/ - creating new branch
git merge /branch name/- used for merging branch with master branch
