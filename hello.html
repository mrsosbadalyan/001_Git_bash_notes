Git DEMO PAGE

Here are the most used commands in gitbash

THIS ARE MOST GENERAL COMMANDS

cd /FolderName/ - entering a folder
cd .. - exiting a folder
git init - initializing a repository
touch /Filename.Filetype/ - creating files ls - shows the files and folders
ls -a - shows hidden files and folders
git status - checks if there are files that can be commited
git add /Filename.Filetype/ - adds the file we want
git add . - adds all files in the list git commit -m "Comment here what changes are done" git log - see previous saves
q - exit after git log
git clear - clear all commands in gitbash

THIS COMMANDS ARE USED TO CREATE BRANCHES, WORK ON A SEPERATE CODE, THEN COME BACK TO PREVIOUS VERSION ON GLOBAL PART OR COMBAIN

git checkout /OldVersionCode/ - returns the previous versions
git checkout /OldVersionCode/ -f - force return if no working
git branch - shows the branches in list
git checkout master - taking back to master
git branch /NewBranchName/ - creating new branch
git checkout /BranchName/ - goes out from the branch git merge /branch name/- used for merging branch with master branch

THIS PART IS USED FOR GITHUB LINKING

git remote -v - checks whether I have already existing connection git remote add origin /UrlFromGitHub/ - linking with url git push -u origin master - pushing it to GITHUB(later this will be used to push latest changes as well) git pull origin master - pulling code fro GITHUB
